# usage

## nsfw_data_scraper图片下载器

是这里面比较完善而且比较有用的一个  
[nsfw_data_scraper](https://github.com/alexanderkim-io/nsfw_data_scraper/tree/master/raw_data) 是一个特殊的测试图片库  
你可能会使用它做些特殊的事情, 那么你就需要这个工具了  

```bash
python RawLoader.py
```
